# 📸 G&L Studio - Gestione Clienti

Applicazione web per la gestione di clienti, preventivi e archivio fotografico.

## 🚀 Come usare l'applicazione

### Primo Avvio

1. **Apri il file `index.html`** nel tuo browser (Chrome, Edge, Safari).
2. **Login**:
   - Username: `G&LStudio`
   - Password: `12763Mlg@`

### Importa i Tuoi Dati Esistenti

1. Apri il file `import-backup.html` nel browser
2. Clicca sulla zona di upload e seleziona il tuo file `gl-studio-clienti.json`
3. Attendi la conferma di importazione
4. Verrai reindirizzato automaticamente alla Dashboard

### Utilizzo Quotidiano

- **Dashboard**: Panoramica statistica e azioni rapide
- **Nuovo Cliente**: Aggiungi nuovi clienti con tutti i dettagli
- **Archivio**: Cerca, filtra e gestisci tutti i clienti
- **Backup**: Scarica regolarmente i dati per sicurezza

## 💾 Backup Dati

I dati sono salvati nel browser (LocalStorage).

- **Fai regolarmente il backup** usando il pulsante "📥 Backup Dati" nella Dashboard
- Conserva i file `.json` scaricati in un luogo sicuro (es. chiavetta USB o cloud privato)
- Usa `import-backup.html` per ripristinare i dati su un nuovo computer

## ⚠️ Importante

- Questa applicazione funziona **in locale** nel browser
- Se cambi computer o cancelli la cronologia del browser, **perdi i dati** se non hai un backup
- I file JSON con i dati dei clienti **NON** vengono caricati su GitHub per proteggere la privacy
